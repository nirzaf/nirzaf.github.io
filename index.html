Pharmacy Management System

Web Application

Pharmacy Management System has following features!

  • Admin and Pharmacist
  • Admin Module
    1. Dashboard: In this section, admin can see all details in brief like a total pharmacist, total Medical Company, total Medicine, total today’s sale, total yesterday’s sale, total seven day’s sale, and total
    2. Invoice Search: In this section, admin can search invoice through invoice number and mobile number.
    3. Medicine Inventory: In this section, admin can check remaining medicine.
    4. Pharmacy Company: In this section, admin can manage pharmacy company (Add/Update).
    5. Medicine: In this section, admin can manage medicine (Add/Update).
    6. Pharmacist: In this section, admin can manage pharmacist (Add/Update).
    7. Reports: In this section, admin can view total sale according to date, according to the pharmacist and total year-wise sale.
    8. Admin can also update his profile, change the password and recover the password.
  • Pharmacist Module
    1. Dashboard: In this section, pharmacist can view their own sale details in brief like total today’s sale, total yesterdays sale, total seven days sale, total
    2. Inventory: In this section, pharmacist can check remaining medicine.
    3. Search: In this section, pharmacist can search medicine by name and add search medicine to cart.
    4. Invoice Search: In this section, pharmacist can search invoice through invoice number and mobile number.
    5. Sales Report: In this section, pharmacist views their own sold medicine according to date.
    6. Pharmacist can also update his profile, change the password and recover the password.
  • Date: March 2017
  • Client: Akurana Medicals (Pvt) Limited and Some other Pharmacies
  • Category: Pharmacy Management
